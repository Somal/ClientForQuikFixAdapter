# Client for system used FIX protocol

## Requirements:
 - Python 3 (tested on Python 3.4)
 - pip (for installing)
 - TypeChecker (for runtime variable type checking)
 
### Installing
  pip3 install -r requirements
  
  
## Running
 - Run QuikFixAdapter ( setup 9000 as input port)
 - Run client.py
